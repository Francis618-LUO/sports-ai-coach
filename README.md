# 🎾⛳ Sports AI Coach - 双运动AI姿态教练

AI-powered motion analysis for tennis & golf. Upload a photo, get real-time pose detection and personalized coaching feedback.

## Tech Stack
- Frontend: React + TypeScript + Vite
- AI: MediaPipe Pose (browser) + Claude API (coaching)
- Mobile-first responsive design

## Get Started
```bash
npm install
npm run dev
```
